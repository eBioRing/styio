# Support

Use GitHub issues for reproducible compiler, CLI, test, and documentation
problems in this repository.

Before filing:

- confirm the branch and commit
- include the command you ran
- include the smallest `.styio` source or fixture that reproduces the behavior
- note whether the behavior also fails under `build/default/bin/styio`

Questions about package management, hosted services, visual tooling, or external
integration should go to the repository that owns that surface.
