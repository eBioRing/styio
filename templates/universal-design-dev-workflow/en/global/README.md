# Global conventions

**Purpose:** Locale-specific norms for this kit: artifact names, directory parity, SSOT rules, adoption patterns. **Details:** [`ARTIFACT-LAYOUT.md`](./ARTIFACT-LAYOUT.md), [`SSOT-AND-MAINTENANCE-RULES.md`](./SSOT-AND-MAINTENANCE-RULES.md).

The `global/` folder is duplicated under `en/` and `zh-CN/` so each locale tree is self-contained. If a rule conflicts with another file in this tree, **update `global/` first**, then align the rest.
